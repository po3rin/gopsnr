# gopsnr

this package calucurate MSE/PSNR.

## Quick Start

```bash
make run CMD="go run cmd/gopsnr/main.go img1.jpeg img2.jpeg"

compare: img1.jpeg & img2.jpeg
[MSE] value: +4.092628e-002 [PSNR] value: +6.201078e+001
```
